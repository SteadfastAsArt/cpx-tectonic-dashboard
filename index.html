Key Findings

  • 10 experiments across 4 data subsets systematically evaluate how data scope, feature completeness, rock type filtering, and crystal type selection affect classification performance.
  • Best practical model: Excl. trace-missing achieves F1-macro = 0.909 ± 0.005 on 39,631 samples by excluding rows with no trace data at all, verified by 5-fold cross-validation for reliable deployment.
  • Basalt phenocryst target subset: F1-macro = 0.734 ± 0.023 on 9,653 samples. Trace elements improve basalt phenocryst classification from 0.621 to 0.734 (+11.3 percentage points), confirming their discriminative value even in this restricted subset.
  • M2_T0 achieves 97.2% F1-macro using complete trace element data, but only 1,195 samples are available — demonstrating strong discriminative signal in trace elements when present.
  • 409.8:1 class imbalance between Intraplate Volcanics and Ocean-Basin Flood Basalt requires careful sample weighting or resampling strategies across all subsets.

Key Insight

  • Basalt clinopyroxene classification achieves F1-macro = 0.717, substantially lower than the full dataset (0.909). This confirms that rock type diversity carries strong tectonic signal — kimberlites are almost exclusively from Archean Cratons, while basalts span all tectonic environments with overlapping geochemistry.
